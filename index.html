Теоретическая модель

МЕТОД ОЦЕНКИ УСТОЙЧИВОСТИ СООРУЖЕНИЙ ИЗ ЗАПОЛНЕННЫХ ОБОЛОЧЕК, УСТАНАВЛИВАЕМЫХ НА СЖИМАЕМОМ ОСНОВАНИИ

Метод разработан для предварительной оценки устойчивости оболочечных сооружений из условия обеспечения несущей способности основания. (Подробно математическая модель данного метода описана в работе Цимбельмана и др. [1]).

Теоретическая модель оценки устойчивости грунтовых оснований рассматриваемых сооружений основана на решении задачи о предельном напряженном состоянии грунтового основания с учетом развития полей скольжения. Исходя из предельной несущей способности основания, определяется предельная нагрузка, действующая на оболочку, при которой система «оболочка-основание» сохраняет состояние равновесия.

Предложенная зависимость представляет собой теоретическую модель, функционально связывающую параметры грунта основания и размеры сооружения с максимальной горизонтальной силой, которую сооружение воспринимает без потери несущей способности основания.

Модель позволяет осуществить быстрый предварительный подбор характеристик основания при заданных габаритах сооружения из условия сохранения несущей способности грунта, либо при заданных прочностных характеристика грунта определяется предельная горизонтальная сила, прикладываемая к оголовку сооружения заданных габаритов.
