Теоретическая модель

Тема исследования

Для оценки напряжённо-деформированного состояния цилиндрической оболочки с наполнителем, установленной на сжимаемом грунтовом основании, была разработана теоретическая модель оценки устойчивости грунтовых оснований оболочечных гравитационных сооружений.

Разрабатываемая теоретическая модель оценки устойчивости грунтовых оснований рассматриваемых сооружений, основана на решении задачи о предельном напряжённом состоянии грунтового основания, с учётом развития полей скольжения. Исходя из предельной несущей способности основания, определяется предельная нагрузка на оболочку, при которой система «оболочка-основание» сохраняет состояние равновесия. Предложенная зависимость представляет собой теоретическую модель, функционально связывающую параметры грунта основания и размеры сооружения с максимальной горизонтальной силой, которую сооружение воспринимает без потери несущей способности основания. Модель позволяет осуществить быстрый предварительный подбор характеристик основания при заданных габаритах сооружения из условия сохранения несущей способности грунта.
